Pokémon App - Pokedex
Una aplicación interactiva diseñada para la exploración y consulta de información detallada sobre diferentes especies de Pokémon. El proyecto se enfoca en la visualización clara de datos técnicos y una navegación fluida para el usuario.

Características
Visualización de catálogo de Pokémon.

Interfaz funcional desarrollada con tecnologías web modernas.

Diseño optimizado para diferentes resoluciones de pantalla.

Transparencia en los Datos y Uso de IA
Se hace constar que se ha utilizado Inteligencia Artificial en el desarrollo y estructuración de secciones específicas del proyecto. La IA fue empleada para organizar, redactar y estandarizar la información correspondiente a:

Estadísticas base: Normalización y formateo de los valores de combate.

Habilidades: Redacción de descripciones y efectos especiales.

Movimientos: Estructuración de datos técnicos y categorías de ataque.

El uso de esta tecnología permitió asegurar que el volumen de datos mostrado mantuviera una consistencia técnica y gramatical en toda la aplicación.

Tecnologías utilizadas
Lenguajes: HTML, CSS y JavaScript.

Fuente de datos: PokeAPI.

Procesamiento de datos: Soporte mediante modelos de lenguaje (IA).
